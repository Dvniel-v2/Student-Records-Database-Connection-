# UniRecords Documentation

Read these files in this order if you are new to the project:

1. `start-here.md`
   Install PostgreSQL, create the database, configure `.env`, run Flask and
   verify the browser pages.

2. `how-unirecords-works.md`
   Understand the browser flow, routes, services, repositories and PostgreSQL
   connection.

3. `database-and-data.md`
   Understand `use_records`, `use_record_management`, the approved SQL scripts,
   Student withdrawal behaviour and report query logic.

4. `assignment-requirements.md`
   Match assignment requirements to implemented routes, services, repositories
   and database objects.

5. `final-submission-checklist.md`
   Track quality checks, screenshots, security checks and final package review.
